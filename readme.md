## Software Requirements
  - Docker
  - Node.js 18+
  - Fast Internet :D

## How to run project
#### Start run this docker command

```docker compose up -d```
or ```yarn start```
 - After that you can enter link below to play.
 - Enjoy xD.

#### URL
>[http://localhost:5173](http://localhost:5173)

To stop run application, just run this command
```docker compose down```
